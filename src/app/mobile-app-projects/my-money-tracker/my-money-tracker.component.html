MyMoneyTracker App

The MyMoneyTracker App is a Kotlin-based mobile application designed to help users effectively manage their personal finances. It allows income and expense tracking, net worth visualization via bar charts, and a detailed transaction history stored with RecyclerView. Users can add transactions with amount, type, date, and description, while a pie chart breakdown and financial tips provide deeper insights.

The app integrates a Flask backend, Microsoft Azure SQL Database, and a tuned LLaMA-3.1-8B model to deliver real-time analytics and guidance. It demonstrates skills in mobile app development, cloud integration, RESTful APIs, and applied AI.
